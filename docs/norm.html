slideflow.norm¶

Stain normalization methods, including both OpenCV (individual image) and Tensorflow/PyTorch (vectorized) implementations.

class slideflow.norm.StainNormalizer(method: str = 'reinhard', source: Optional[str] = None)¶

Supervises stain normalization for images and efficiently convert between common image types.

__init__(method: str = 'reinhard', source: Optional[str] = None) → None¶

Initializer. Establishes normalization method.

Parameters

  • method (str) – Either ‘macenko’, ‘reinhard’, or ‘vahadane’. Defaults to ‘reinhard’.

  • source (str) – Path to source image for normalizer. If not provided, defaults to slideflow.norm.norm_tile.jpg

fit(*args: Optional[Union[slideflow.dataset.Dataset, numpy.ndarray, str]], target_means: Optional[Union[List[float], numpy.ndarray]] = None, target_stds: Optional[Union[List[float], numpy.ndarray]] = None, stain_matrix_target: Optional[Union[List[float], numpy.ndarray]] = None, target_concentrations: Optional[Union[List[float], numpy.ndarray]] = None, batch_size: int = 64, num_threads: Union[str, int] = 'auto') → None¶

Fit the normalizer.

Parameters

  • target_means (Optional[np.ndarray], optional) – Target means. Defaults to None.

  • target_stds (Optional[np.ndarray], optional) – Target stds. Defaults to None.

  • stain_matrix_target (Optional[np.ndarray], optional) – Target stain matrix. Defaults to None.

  • target_concentrations (Optional[np.ndarray], optional) – Target concentrations. Defaults to None.

  • batch_size (int, optional) – Batch size during fitting, if fitting to dataset. Defaults to 64.

  • num_threads (Union[str, int], optional) – Number of threads to use during fitting, if fitting to a dataset. Defaults to ‘auto’.

Raises

  • NotImplementedError – If attempting to fit Dataset using

  • non-vectorized normalizer. –

  • errors.NormalizerError – If unrecognized arguments are provided.
